# EXP-2-PROMPT-ENGINEERING-
## Aim:
Comparative Analysis of different types of Prompting patterns and explain with Various Test Scenarios

## Experiment:
Test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. Analyze the quality, accuracy, and depth of the generated responses.

## Algorithm:
Define Prompt Types

Broad Prompt (vague, unstructured)
Basic Prompt (clear but simple)
Structured Prompt (detailed and specific)
Select Test Scenarios

Concept Explanation
Coding Task
Content Generation
Problem Solving
Design Prompts

Create one broad, one basic, and one structured prompt for each scenario
Generate Responses

Use an AI model to generate outputs for all prompts
Evaluate Responses

Accuracy (correctness)
Clarity (understandability)
Depth (level of explanation)
Relevance (alignment with prompt)
Compare Results

Create comparison tables for each scenario
Analyze Findings

Identify which prompting pattern performs best
## Result
Structured prompts produced the most accurate, clear, and detailed responses
Basic prompts gave correct but less detailed outputs
Broad prompts resulted in vague and low-quality responses
Overall Comparison
Criteria	Broad	Basic	Structured
Accuracy	❌ Low	⚠️ Medium	✅ High
Clarity	❌ Poor	⚠️ Average	✅ Excellent
Depth	❌ Low	⚠️ Medium	✅ High
Usefulness	❌ Low	⚠️ Medium	✅ High
Final Insight
Better prompt design directly improves the quality of AI-generated responses.
